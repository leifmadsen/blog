.bashrc trick for git repo and branch information

Dec 22, 2011 · 1 minute read · Comments
TechnologyUseful Tools The other day I was talking to my friend Russell Bryant who pointed me to some .bashrc magic that would show me which branch I was currently working with inside a git repo on my system. I found it incredibly handy and have modified the ANSI colour coding slightly. export PS1='[\u@\h \[33[0;36m\]\W$(__git_ps1 "\[33[0m\]\[33[0;33m\](%s)")\[33[0m\]]\$ ' On Fedora Russell mentioned that you need the bash-completion installed.
